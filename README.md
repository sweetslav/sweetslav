## Текущий проект (главный на сегодня)

### **Wildberries Data Parser (Google Apps Script)**
Это мой ключевой проект, который я разрабатываю для ежедневной работы.

**Задача:** Автоматически собирать данные по заказам, рекламным расходам, кликам и конверсиям из нескольких кабинетов Wildberries и загружать их в Google Sheets для анализа эффективности.

**Стек:** Google Apps Script, Wildberries API, Google Sheets API.
**Особенности:**
- Работает с 3 разными API Wildberries (Статистика, Аналитика, Продвижение).
- Собирает и агрегирует данные по артикулам (nmId).
- Считает ключевые метрики: ДРР (офлайн), CR1.
- Использует пагинацию для обработки >1000 товаров.
- Поддерживает несколько кабинетов через единую конфигурацию.
- Полностью развернут в облаке Google, не требует локального сервера.

**Результат:** Сотни часов сэкономленного времени и возможность принимать решения на основе актуальных данных каждое утро.

- [Ссылка на проект](https://github.com/sweetslav/wb-parser)
